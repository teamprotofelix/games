# 🎮 신나는 게임월드 (Exciting Game World)

초등학생부터 중학생까지 즐길 수 있는 정적 게임 홈페이지입니다. GitHub Pages에서 바로 사용할 수 있습니다.

## 📁 프로젝트 구조

```
/
├── index.html          # 메인 페이지
├── css/
│   └── style.css       # 모든 스타일 (Mobile-First)
├── js/
│   └── script.js       # 앱 컨트롤러 + 16가지 게임
├── images/             # 이미지 폴더 (아래 이미지들을 넣어주세요)
│   ├── herologo.jpg    # 헤더 로고 (원형, 200x200px 권장)
│   └── hero-character.png  # 메인 캐릭터 이미지 (200x200px 권장)
└── README.md
```

## 🖼️ 필요한 이미지 파일

아래 이미지 파일들을 `images/` 폴더에 넣어주세요. 이미지가 없어도 사이트는 이모지로 정상 작동합니다.

| 파일명 | 용도 | 권장 크기 |
|--------|------|-----------|
| `images/herologo.jpg` | 상단 헤더 로고 | 200×200px (원형) |
| `images/hero-character.png` | 메인 화면 캐릭터 | 200×200px |
| `images/bg-pattern.png` | 배경 패턴 (선택) | 타일 가능한 패턴 |
| `images/card-bg-1.png` ~ `card-bg-16.png` | 게임 카드 배경 (선택) | 300×200px |
| `images/favicon.png` | 파비콘 (선택) | 32×32px |

## 🚀 사용 방법

### GitHub Pages에 배포
1. 이 폴더를 GitHub 저장소에 Push
2. Settings → Pages → Source를 `main` 브랜치로 설정
3. Save 후 제공된 URL로 접속

### 로컬에서 실행
그냥 `index.html` 파일을 브라우저에서 열면 됩니다.

## 📱 특징

- **정적 사이트** - HTML/CSS/JS만 사용, 서버 불필요
- **모바일 최적화** - 스마트폰 화면에 맞춘 디자인
- **터치 지원** - 모든 게임 터치/스와이프 완벽 지원
- **16가지 게임** - 다양한 장르의 게임을 한 곳에서
- **접근성** - 키보드(ESC로 뒤로가기, 방향키) 지원
- **오프라인 가능** - 인터넷 연결 없이도 게임 플레이 가능
